\subsubsection{Iterative Statements}
Languages like Python or Java have two loops: \cd{for} loop for iterating within a given range and \cd{while} loop for continuous iteration \il{while} a given condition is true. TaylorScript combines these into one looping construct \cd{AllTooWhile}:
\begin{minted}{text}
-- <header> : <initialization> ; <condition> ; <updation>
AllTooWhile [<header> | <condition>] [
    <body>
]
\end{minted}
\bd{Example 1:}
\begin{minted}{text}
AllTooWhile [i = 0; i <= 10; ++i] [
    When [i % 2 == 0] [
        SpeakNow[i];
    ]
]
\end{minted}
Output:
\begin{minted}{text}
    0
    2
    4
    6
    8
    10
\end{minted}
\bd{Example 2:}
\begin{minted}{text}
AllTooWhile [x=0; x < 3; x++] [
    SpeakNow[x];
    AllTooWhile [y=0; y < 3; y++] [
        SpeakNow[y];
    ]
]
\end{minted}
Output:
\begin{minted}{text}
0
0
1
2
1
0
1
2
2
0
\end{minted}
